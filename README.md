# NetDuel

## Overview
**NetDuel** is a **Rock-Paper-Scissors** game playable in both **single-player and multiplayer modes** directly from the **Command Line Interface (CLI)**.  

- Play against an **AI opponent** or other players online.  
- Built with **Python, Go, and Node.js**, showcasing a full-stack CLI application.

---

## Features
- Single-player vs AI  
- Multiplayer online mode  
- CLI-based gameplay for simplicity and speed  
- Cross-language architecture using Python, Go, and Node.js

## Installation

### Prerequisites
Ensure the following are installed:
- **Python**
- **Golang**
- **Node.js**

### Steps
1. Clone the repository:
	```bash
	git clone https://github.com/EJ-Edwards/NetDuel.git
	cd NetDuel
	```
2. Install Node.js dependencies:
	```bash
	cd cli-dash
	npm install
	cd ..
	```
3. (Optional) Set up a Python virtual environment and install dependencies if needed.
4. (Optional) Install Go dependencies if needed.

## Usage

### Start the CLI
```bash
cd cli-dash
node index.js
```

### Single Player
- Select **Single Player** from the menu to play against the AI.

### Multiplayer
- Select **Multiplayer** from the menu.
- Choose to **create** a room (share the pin) or **join** a room (enter the pin).
- Play with friends by sharing the room pin.

---

## Architecture
- **cli-dash/**: Node.js CLI interface
- **engine/**: Python game logic (AI, rules)
- **server/**: Go multiplayer server (WebSocket)
