## NetDuel Documentation
# Overview
NetDuel is a multiplayer and single player Rock-Paper-Scissors Game that can be installed and played on the Command Line Interface. Its features are that it uses an AI and it plays against you.

## Installation
 1. How to install this is first by making sure you have Python, Golang and Node.js installed 
 2. First use copy and paste this "git clone https://github.com/EJ-Edwards/NetDuel.git" (note remove the quotation marks or else this instaliaton wont work)
 3. cd into cli-dash and than run node index.js
 4. From there you can have fun and play this game
